This extension displays detailed information about the current website. Including a shortcut to Your public IP Address (myIP info). The information is grouped in the following tabs:

IPv4: shows the IP Address Information (IPv4) for the current website.

IPv6: shows the IP Address Information (IPv6) for the current website.

ISP: shows the ISP information for the current website.

Domain: shows information about the domain of the current website.

My IP: shows information about your public IP address.
